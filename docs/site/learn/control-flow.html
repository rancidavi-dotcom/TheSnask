Capítulo 3

Controle de Fluxo

Snask oferece estruturas de controle familiares, mas com um foco rigoroso em escopo e segurança.

Condicionais: if e else

No Snask, as condicionais não exigem parênteses ao redor da expressão, seguindo a tendência de linguagens modernas como Go e Rust. 

let score = 85

if score >= 90 {
    print("A - Excelente!\n")
} else if score >= 70 {
    print("B - Bom trabalho\n")
} else {
    print("C - Precisa melhorar\n")
}

Loops: while

O Snask simplifica a iteração oferecendo apenas o loop while como primitiva básica. Não existe o loop for(i=0; i<10; i++) clássico de C, o que reduz erros de "off-by-one". 

mut i = 0
while i < 5 {
    print("Iteração: {i}\n")
    i = i + 1
}

Iteração com Coleções

Para iterar sobre listas, usamos a função builtin range() combinada com o while, mantendo o controle total sobre o índice. 

let lista = ["A", "B", "C"]
mut idx = 0
while idx < len(lista) {
    print("Item: {lista[idx]}\n")
    idx = idx + 1
}

Estilos de Bloco: Chaves vs Indentação

Uma característica única do Snask é que ele aceita tanto chaves {} quanto indentação significativa (estilo Python).

Estilo Chaves (Recomendado para tutoriais):

if true {
    print("Bloco com chaves\n")
}

Estilo Indentação:

if true:
    print("Bloco por indentação\n")
Dica Profissional: Escolha um estilo para o seu projeto e mantenha a consistência. O compilador Snask é capaz de detectar misturas inconsistentes e emitir diagnósticos de ajuda.
