Guia rapido

Comecando com Snask

Este guia leva do zero ate um programa compilado. Ele usa apenas superficie que combina com o compilador atual.

Instalar

curl -fsSL https://raw.githubusercontent.com/rancidavi-dotcom/TheSnask/main/install.sh | bash
export PATH="$HOME/.snask/bin:$PATH"
snask doctor

O instalador detecta distros Linux comuns, instala dependencias quando possivel e encontra LLVM 18 para o backend atual.

Primeiro arquivo

Crie `hello.snask`:

class main {
    fun start() {
        print("Ola, Snask!\n")
    }
}

Compilar e rodar

snask build hello.snask --output hello
./hello

Sem `--profile`, Snask usa `humane`: runtime completo, mensagens de erro humanas e uma experiencia mais amigavel.

Quando algo der errado

snask explain S1005

Diagnosticos publicos usam codigos curtos, spans, snippets e ajuda acionavel. Ainda ha mensagens em ingles em algumas partes, mas o formato publico ja existe.
