Basico da linguagem

Snask aceita blocos por chaves e tambem por indentacao. Para tutoriais copiaveis, usamos chaves.

Variaveis

class main {
    fun start() {
        let name = "Davi"
        mut age = 25
        age = age + 1
        print("Nome: {name}\n")
        print("Idade: {age}\n")
    }
}

Condicionais e loops

class main {
    fun start() {
        let score = 8

        if score >= 7 {
            print("Aprovado\n")
        } else {
            print("Reprovado\n")
        }

        mut i = 0
        while i < 3 {
            print("Passo: {i}\n")
            i = i + 1
        }
    }
}

Funcoes tipadas simples

fun add(a: int, b: int) : int {
    return a + b
}

class main {
    fun start() {
        let value: int = add(20, 22)
        print("Resultado: {value}\n")
    }
}

Colecoes tipadas

class main {
    fun start() {
        let fruits: list<str> = ["maca", "banana", "uva"]
        let scores: dict<str, int> = { "alice": 10, "bob": 20 }

        print(fruits[0])
        print("\n")
        print(scores["alice"])
        print("\n")
    }
}

Classes nominais simples

class Animal {
    let age: int = 0
}

class Dog extends Animal {
    let name: str = "rex"
}

fun show_age(animal: Animal) : int {
    return animal.age
}

class main {
    fun start() {
        let dog: Dog = new Dog()
        let age: int = show_age(dog)
        print("Idade: {age}\n")
    }
}
Classes simples, heranca simples, list<T>, dict<K, V> e funcoes tipadas simples ja tem testes semanticos dedicados. Recursos avancados continuam documentados como parciais.
