Capítulo 4

Funções

Funções são as unidades fundamentais de lógica no Snask. Elas são desenhadas para serem puras, testáveis e explicitamente tipadas.

Declaração Básica

Uma função é declarada usando a palavra-chave fun, seguida pelo nome, parâmetros entre parênteses e o corpo da função. 

fun saudar() {
    print("Olá de uma função!\n")
}

class main {
    fun start() {
        saudar()
    }
}

Parâmetros e Retorno

No Snask profissional, recomendamos sempre anotar os tipos dos parâmetros e o tipo de retorno. Isso permite que o compilador otimize o código via LLVM e detecte erros antes mesmo de você rodar o programa. 

fun somar(a: float, b: float) -> float {
    return a + b
}

class main {
    fun start() {
        let total = somar(10.5, 2.5)
        print("Soma: {total}\n")
    }
}

Escopo e Closures

Variáveis declaradas dentro de uma função são locais àquela função. O Snask gerencia o ciclo de vida dessas variáveis automaticamente usando o sistema de **Zonas**, garantindo que a memória seja liberada assim que a função termina.

Módulos e Organização

Funções podem ser organizadas em arquivos separados e importadas. Isso permite a criação de bibliotecas reutilizáveis e código limpo. 

// math_utils.snask
fun dobrar(n: float) -> float {
    return n * 2
}

// main.snask
import "math_utils"
class main {
    fun start() {
        let x = math_utils::dobrar(5)
    }
}
