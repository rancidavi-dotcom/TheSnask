http_get

Executa requisicao HTTP GET pelo runtime.

experimental 🛡️ humane segura ✅ Segura — não pode corromper memória ou causar UB.

Assinatura

http_get(url: str) -&amp;amp;amp;amp;amp;amp;gt; dict

Para que serve

Realiza uma requisição HTTP do tipo GET para a URL especificada.

Retorna um dicionário contendo o corpo da resposta, o código de status e os cabeçalhos recebidos.

É a ferramenta padrão para consumir dados de APIs REST externas.

Perfil

🛡️ humane — Gerenciamento automático de memória via zonas. Seguro por padrão.

Segurança: ✅ Segura — não pode corromper memória ou causar UB.

Parâmetros

NomeDescrição
urlURL absoluta do recurso.

Retorno

Dicionario com status, headers e body.

Exemplo

let res = http_get(&amp;amp;amp;amp;amp;amp;quot;https://api.github.com/zen&amp;amp;amp;amp;amp;amp;quot;)
print(&amp;amp;amp;amp;amp;amp;quot;Status: {res[\&amp;amp;amp;amp;amp;amp;quot;status\&amp;amp;amp;amp;amp;amp;quot;]}\n&amp;amp;amp;amp;amp;amp;quot;)
print(&amp;amp;amp;amp;amp;amp;quot;Body: {res[\&amp;amp;amp;amp;amp;amp;quot;body\&amp;amp;amp;amp;amp;amp;quot;]}\n&amp;amp;amp;amp;amp;amp;quot;)
