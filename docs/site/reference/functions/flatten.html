flatten

Achata uma lista de listas.

parcial 🛡️ humane segura ✅ Segura — não pode corromper memória ou causar UB.

Assinatura

flatten(values: list) -&amp;amp;amp;amp;gt; list

Para que serve

flatten transforma uma lista que contém sublistas em uma única lista plana.

É amplamente usada em processamento de dados para normalizar estruturas aninhadas, como resultados de consultas, grids ou agrupamentos.

Perfil

🛡️ humane — Gerenciamento automático de memória via zonas. Seguro por padrão.

Segurança: ✅ Segura — não pode corromper memória ou causar UB.

Parâmetros

NomeDescrição
valuesLista contendo sublistas.

Retorno

Uma nova lista com todos os elementos das sublistas concatenados em ordem.

Exemplo

let xs = flatten([[1], [2]])

Cuidados e Edge Cases

  • API parcial: pode ter limites com tipos muito aninhados.
  • Achata apenas um nível de profundidade (shallow flatten).
