snaskgui_key_down

Consulta estado de tecla.

experimental perfil: systems seguranca: segura; buffers usam @unsafe quando houver ptr

Assinatura

snaskgui_key_down(window: any, key: float) -> bool

Para que serve

`snaskgui_key_down` faz parte da API framebuffer da Snask. Essa familia e pensada para programas que controlam pixels diretamente: emuladores, jogos simples, visualizadores binarios e ferramentas de renderizacao deterministica.

Consulta estado de tecla. Em vez de montar uma interface com botoes e layouts, voce manipula uma janela e envia frames prontos para ela.

Parametros

NomeDescricao
windowHandle da janela.
keyCódigo numérico da tecla.

Retorno

Conforme assinatura.

Exemplo

import "snaskgui"

class main {
    fun start() {
        snaskgui::init()
        let win = snaskgui::window("Pixels", 256, 240, 2)
        snaskgui::poll(win)
        snaskgui::close(win)
    }
}
