gui

gui_entry

Cria input de texto.

experimental perfil: humane seguranca: segura

Assinatura

gui_entry() -> any

Para que serve

`gui_entry` pertence a camada GUI baseada em widgets nativos. Ela e experimental e serve para construir janelas, labels, botoes, entradas e containers sem sair da sintaxe Snask.

Cria input de texto. Handles retornados por funcoes GUI sao opacos: passe-os para outras funcoes `gui_*`, mas nao tente interpretar o valor manualmente.

Parametros

NomeDescricao
Sem parametros.

Retorno

Conforme assinatura.

Exemplo

import "gui"

class main {
    fun start() {
        let ok = gui::init()
        let win = gui::window("Snask", 320, 180)
        gui::show_all(win)
        gui::run()
    }
}

Cuidados

  • API experimental: nome, retorno ou comportamento ainda podem mudar conforme o runtime amadurece.
