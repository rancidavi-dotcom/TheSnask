gui

gui_entry

Cria input de texto.

experimental 🛡️ humane segura ✅ Segura — não pode corromper memória ou causar UB.

Assinatura

gui_entry() -&amp;amp;amp;amp;amp;amp;gt; any

Para que serve

`gui_entry` pertence a camada GUI baseada em widgets nativos. Ela e experimental e serve para construir janelas, labels, botoes, entradas e containers sem sair da sintaxe Snask.

Cria input de texto. Handles retornados por funcoes GUI sao opacos: passe-os para outras funcoes `gui_*`, mas nao tente interpretar o valor manualmente.

Perfil

🛡️ humane — Gerenciamento automático de memória via zonas. Seguro por padrão.

Segurança: ✅ Segura — não pode corromper memória ou causar UB.

Parâmetros

NomeDescrição
Nenhum parâmetro.

Retorno

Conforme assinatura.

}
