lo_u8

Extrai o byte baixo de uma word.

estável 🛡️ humane segura ✅ Segura — não pode corromper memória ou causar UB.

Assinatura

lo_u8(value: any) -&amp;amp;amp;amp;amp;amp;gt; u8

Para que serve

Extrai o byte menos significativo (Low Byte) de um valor de 16 bits ou maior.

Em sistemas Little Endian (como o Snask), este é o primeiro byte da representação em memória.

Perfil

🛡️ humane — Gerenciamento automático de memória via zonas. Seguro por padrão.

Segurança: ✅ Segura — não pode corromper memória ou causar UB.

Parâmetros

NomeDescrição
valueValor de 16 bits ou mais.

Retorno

Retorna `u8`.

Exemplo

let word = 0xABCD
let lo = lo_u8(word) // 0xCD
