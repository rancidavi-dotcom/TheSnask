carry_add_u8

Calcula carry de soma de 8 bits, incluindo carry de entrada.

estável 🛡️ humane segura ✅ Segura — não pode corromper memória ou causar UB.

Assinatura

carry_add_u8(a: any, b: any, carry_or_borrow: any) -&amp;gt; bool

Para que serve

Calcula o bit de carry (transporte) resultante de uma soma de 8 bits. É fundamental para implementar somas de precisão arbitrária (ex: somar dois números de 16 bits usando operações de 8 bits).

A função leva em conta o bit de carry de entrada, simulando exatamente o comportamento da instrução `ADC` (Add with Carry) de processadores como o 6502.

Perfil

🛡️ humane — Gerenciamento automático de memória via zonas. Seguro por padrão.

Segurança: ✅ Segura — não pode corromper memória ou causar UB.

Parâmetros

NomeDescrição
aOperando A.
bOperando B.
carryCarry de entrada (0 ou 1).

Retorno

Booleano da condicao.

Exemplo

// Soma 255 + 1 com carry de entrada 0
let carry = carry_add_u8(255, 1, 0)
// carry será true, pois 255 + 1 estoura 8 bits.
