Assinatura

wrapping_inc(value: any) -> any

Para que serve

`wrapping_inc` e uma primitiva deterministica para sistemas. Ela foi criada para casos como CPU 6502, parsers binarios e logica que precisa controlar bits, overflow ou inteiros de tamanho fixo.

Incrementa com wrap. A ideia e deixar o comportamento explicito, sem depender de conversoes magicas.

Parametros

NomeDescricao
valueValor base.

Retorno

Retorna `any`.

Exemplo

let x: u8 = wrapping_inc(255, 1)
