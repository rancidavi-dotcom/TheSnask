overflow_sub_i8

Calcula overflow assinado de subtracao de 8 bits.

estavel perfil: systems seguranca: segura

Assinatura

overflow_sub_i8(a: any, b: any, carry_or_borrow: any) -> bool

Para que serve

`overflow_sub_i8` e uma primitiva deterministica para sistemas. Ela foi criada para casos como CPU 6502, parsers binarios e logica que precisa controlar bits, overflow ou inteiros de tamanho fixo.

Calcula overflow assinado de subtracao de 8 bits. A ideia e deixar o comportamento explicito, sem depender de conversoes magicas.

Parametros

NomeDescricao
aMinuendo.
bSubtraendo.
borrowBorrow de entrada.

Retorno

Booleano da condicao.

Exemplo

let flag = overflow_sub_i8(0xFF, 0x01, 0)
