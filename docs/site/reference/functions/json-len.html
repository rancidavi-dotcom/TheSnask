json_len

Retorna tamanho de array/objeto JSON.

experimental 🛡️ humane segura ✅ Segura — não pode corromper memória ou causar UB.

Assinatura

json_len(value: any) -&amp;gt; float

Para que serve

`json_len` pertence a uma biblioteca nativa importavel. Em codigo Snask normal, prefira chamar pelo modulo, por exemplo `json::parse` ou `sfs::read`, em vez do nome interno com prefixo.

Retorna tamanho de array/objeto JSON. A assinatura documenta a superficie que o analisador conhece hoje.

Perfil

🛡️ humane — Gerenciamento automático de memória via zonas. Seguro por padrão.

Segurança: ✅ Segura — não pode corromper memória ou causar UB.

Parâmetros

NomeDescrição
valueObjeto ou Array JSON.

Retorno

Conforme assinatura.

Exemplo

import &amp;quot;json&amp;quot;

class main {
    fun start() {
        let obj = json::parse(&amp;quot;{\&amp;quot;name\&amp;quot;:\&amp;quot;snask\&amp;quot;}&amp;quot;)
        print(json::get(obj, &amp;quot;name&amp;quot;))
    }
}
