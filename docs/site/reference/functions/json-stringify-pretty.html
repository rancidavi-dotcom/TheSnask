json

json_stringify_pretty

Serializa valor dinamico para JSON formatado.

experimental perfil: humane seguranca: segura

Assinatura

json_stringify_pretty(value: any) -> str

Para que serve

`json_stringify_pretty` pertence a uma biblioteca nativa importavel. Em codigo Snask normal, prefira chamar pelo modulo, por exemplo `json::parse` ou `sfs::read`, em vez do nome interno com prefixo.

Serializa valor dinamico para JSON formatado. A assinatura documenta a superficie que o analisador conhece hoje.

Parametros

NomeDescricao
valueValor a ser serializado com indentação.

Retorno

Conforme assinatura.

Exemplo

import "json"

class main {
    fun start() {
        let obj = json::parse("{\"name\":\"snask\"}")
        print(json::get(obj, "name"))
    }
}

Cuidados

  • API experimental: nome, retorno ou comportamento ainda podem mudar conforme o runtime amadurece.
