saturating_add

Soma saturando no limite do tipo.

estavel perfil: systems seguranca: segura

Assinatura

saturating_add(a: any, b: any) -> any

Para que serve

`saturating_add` e uma primitiva deterministica para sistemas. Ela foi criada para casos como CPU 6502, parsers binarios e logica que precisa controlar bits, overflow ou inteiros de tamanho fixo.

Soma saturando no limite do tipo. A ideia e deixar o comportamento explicito, sem depender de conversoes magicas.

Resultado saturado.
