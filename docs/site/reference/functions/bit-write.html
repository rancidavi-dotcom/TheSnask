bit_write

Liga ou desliga bit conforme booleano/valor.

estável 🛡️ humane segura ✅ Segura — não pode corromper memória ou causar UB.

Assinatura

bit_write(value: any, bit: any, enabled: any) -&amp;gt; any

Para que serve

Define o estado de um bit específico (0 ou 1) baseado em um valor booleano ou numérico.

É uma forma declarativa de modificar flags sem precisar lidar manualmente com operadores de máscara (`|` e `&`).

Perfil

🛡️ humane — Gerenciamento automático de memória via zonas. Seguro por padrão.

Segurança: ✅ Segura — não pode corromper memória ou causar UB.

Parâmetros

NomeDescrição
valueValor base.
bitIndice do bit.
enabledBooleano ou 0/1.

Retorno

Retorna `any`.

Exemplo

mut status = 0b00000000
let condicao = true
status = bit_write(status, 7, condicao)
// status agora é 0b10000000
