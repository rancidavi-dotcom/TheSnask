wrapping_add

Soma com overflow por mascara do tipo, util para CPUs reais.

estável 🛡️ humane segura ✅ Segura — não pode corromper memória ou causar UB.

Assinatura

wrapping_add(a: any, b: any) -&amp;amp;amp;amp;gt; any

Para que serve

Realiza uma soma aritmética que ignora o estouro (overflow). Se o resultado for maior que o valor máximo suportado pelo tipo, ele 'dá a volta' (wrap around) para zero.

Esta é a semântica padrão de CPUs reais e é obrigatória para a implementação correta de lógica de registradores de CPU (como o Program Counter ou o acumulador de um 6502/Z80).

Perfil

🛡️ humane — Gerenciamento automático de memória via zonas. Seguro por padrão.

Segurança: ✅ Segura — não pode corromper memória ou causar UB.

Parâmetros

NomeDescrição
aOperando.
bOperando.

Retorno

Resultado com wrap.

Exemplo

// Em um u8, o valor máximo é 255.
let a: u8 = 255
let b: u8 = 1
let res = wrapping_add(a, b)
// res será 0, e não um erro de overflow ou 256.
