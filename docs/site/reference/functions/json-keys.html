json

json_keys

Retorna chaves de objeto JSON.

experimental perfil: humane seguranca: segura

Assinatura

json_keys(value: any) -> any

Parametros

NomeDescricao
value/text/keyValor JSON dinamico, texto de entrada ou chave.

Retorno

Conforme assinatura.

Exemplo

import "json"

class main {
    fun start() {
        let obj = json::parse("{\"name\":\"snask\"}")
        print(json::get(obj, "name"))
    }
}
