snaskgui

snaskgui_init

Inicializa API framebuffer para jogos/emuladores.

experimental 🛡️ humane segura ✅ Segura — não pode corromper memória ou causar UB.

Assinatura

snaskgui_init() -&amp;amp;amp;amp;amp;amp;gt; bool

Para que serve

Inicializa o subsistema de vídeo e eventos do SnaskGUI. Esta deve ser a primeira função chamada antes de qualquer tentativa de criar janelas ou manipular gráficos no perfil systems.

Ela configura os drivers internos (como SDL2 ou DRM, dependendo da plataforma) e prepara o ambiente para renderização de alta performance.

Perfil

🛡️ humane — Gerenciamento automático de memória via zonas. Seguro por padrão.

Segurança: ✅ Segura — não pode corromper memória ou causar UB.

Parâmetros

NomeDescrição
Nenhum parâmetro.

Retorno

Conforme assinatura.

Exemplo

import &amp;amp;amp;amp;amp;amp;quot;snaskgui&amp;amp;amp;amp;amp;amp;quot;

    fun start() {
        if snaskgui::init() {
            print(&amp;amp;amp;amp;amp;amp;quot;Sistema gráfico pronto!\n&amp;amp;amp;amp;amp;amp;quot;)
            // Prossiga com snaskgui::window
        } else {
            print(&amp;amp;amp;amp;amp;amp;quot;Erro ao inicializar gráficos.\n&amp;amp;amp;amp;amp;amp;quot;)
        }
    }
}
