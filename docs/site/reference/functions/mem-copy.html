memory

mem_copy

Copia bytes entre blocos.

estavel perfil: systems seguranca: exige @unsafe

Assinatura

mem_copy(dst: ptr, src: ptr, size: any) -> void

Para que serve

`mem_copy` e uma primitiva de memoria crua do perfil systems. Ela existe para codigo que precisa lidar com bytes, ponteiros e buffers previsiveis.

Esse tipo de funcao pertence a regioes `@unsafe`, porque o compilador nao consegue provar sozinho que offsets, tamanho e tempo de vida estao corretos.

Parametros

NomeDescricao
dstPonteiro de destino.
srcPonteiro de origem.
sizeNumero de bytes a copiar.

Retorno

Nada.

Exemplo

@unsafe {
    mem_copy(dst, src, 256)
}

Cuidados

  • Use dentro de `@unsafe` e mantenha ownership claro.
