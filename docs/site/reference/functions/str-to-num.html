str_to_num

Converte string numerica para numero.

parcial 🛡️ humane segura ✅ Segura — não pode corromper memória ou causar UB.

Assinatura

str_to_num(text: str) -&amp;gt; float

Para que serve

Tenta converter uma representação textual (string) em um valor numérico (float).

Suporta notação decimal e, em alguns runtimes, notação científica.

Perfil

🛡️ humane — Gerenciamento automático de memória via zonas. Seguro por padrão.

Segurança: ✅ Segura — não pode corromper memória ou causar UB.

Parâmetros

NomeDescrição
textTexto numerico.

Retorno

Numero convertido.

Exemplo

let preco_texto = &amp;quot;42.50&amp;quot;
let valor = str_to_num(preco_texto)
let total = valor * 2 // 85.0
