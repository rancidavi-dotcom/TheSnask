sfs_mtime

Tempo de modificacao.

parcial perfil: humane seguranca: segura

Assinatura

sfs_mtime(path: str) -> float

Para que serve

`sfs_mtime` pertence a uma biblioteca nativa importavel. Em codigo Snask normal, prefira chamar pelo modulo, por exemplo `json::parse` ou `sfs::read`, em vez do nome interno com prefixo.

Tempo de modificacao. A assinatura documenta a superficie que o analisador conhece hoje.

Parametros

NomeDescricao
pathCaminho.

Retorno

Conforme assinatura.

Exemplo

import "sfs"

class main {
    fun start() {
        let ok = sfs::exists(".")
        print(ok)
    }
}
