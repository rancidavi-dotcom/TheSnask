mem_read_u8

Le byte em offset.

estável 🛡️ humane segura ✅ Segura — não pode corromper memória ou causar UB.

Assinatura

mem_read_u8(ptr: ptr, offset: any) -&amp;amp;amp;gt; u8

Para que serve

Lê um único byte (8 bits) de um endereço de memória física especificado por um ponteiro base e um deslocamento (offset).

É a primitiva fundamental para implementar barramentos de memória, leitura de cabeçalhos binários e processamento de buffers de rede.

Perfil

🛡️ humane — Gerenciamento automático de memória via zonas. Seguro por padrão.

Segurança: ✅ Segura — não pode corromper memória ou causar UB.

Parâmetros

NomeDescrição
ptrPonteiro base.
offsetDeslocamento em bytes.

Retorno

Valor `u8`.

Exemplo

@unsafe {
    // Lê o byte no offset 0x10 de um buffer
    let status: u8 = mem_read_u8(buffer, 0x10)
    
    if bit_test(status, 7) {
        print(&amp;amp;amp;quot;Flag ativa!\n&amp;amp;amp;quot;)
    }
}
