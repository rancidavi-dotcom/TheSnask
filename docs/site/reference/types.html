Type system

Tipos

O type system da Snask esta ficando mais forte. O nucleo estavel hoje inclui primitivos, colecoes genericas simples e tipos nominais simples.

Primitivos

TipoUso
intInteiro geral.
floatNumero de ponto flutuante.
strString.
booltrue ou false.

Colecoes

let names: list<str> = ["ana", "bia"]
let scores: dict<str, int> = { "ana": 10 }

Nominais

class Animal {
    let age: int = 0
}

class Dog extends Animal {
    let name: str = "rex"
}

let dog: Dog = new Dog()
let animal: Animal = dog

Systems

`u8`, `u16`, `u32`, `u64`, `i8`, `i16`, `i32`, `i64`, `usize`, `isize` e `ptr` existem para baixo nivel. Eles ja sao usados por primitivas systems/NES, mas ainda precisam fechamento completo de layout/ABI.
