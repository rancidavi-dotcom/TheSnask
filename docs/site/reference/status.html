# Parcial ou experimental

Estas areas existem mas ainda nao atingiram contrato estavel. Consulte o estado exato antes de usar.

AreaStatusMotivo
OM formal experimental Arquitetura existe; faltam escape analysis, borrow checking e regras estaticas completas. O runtime de zonas funciona, mas a verificacao estatica ainda e parcial.
C interop universal experimental Funciona para casos seguros (tipos primitivos, ponteiros opacos). Callbacks, structs complexas e ownership ambiguo ainda precisam de patch ou regra explicita.
GUI nativa experimental Runtime existe com janelas, botoes, labels e entrada de texto. API ainda nao e contrato maduro — mudancas quebradiças podem ocorrer.
snaskgui (OpenGL) experimental Polling de eventos, apresentacao de framebuffer RGBA, gerenciamento de janelas. Adequado para experimentos graficos, nao para producao.
LSP / IDE parcial Recursos de diagnostico e parse existem. Falta suite automatizada de protocolo e suporte a completacao de codigo.
import com from parcial from utils import math_helpers funciona para casos simples. Namespaces aninhados e re-export ainda estao sendo definidos.
String interpolation avancada parcial "ola {nome}" funciona. Formatacao numerica (:.2f) e expressoes dentro de chaves ainda nao.

# Como essas classificacoes sao validadas

Cada feature marcada como estavel possui um ou mais testes versionados em docs/examples/reference/ que compilam e passam no CI. Features experimental ou parcial tem testes parciais ou apenas validacao no analisador semantico.

Se voce encontrar uma feature marcada como estavel cujo teste esta quebrado, abra uma issue no repositorio. Honestidade do contrato e prioridade do projeto.

Criterios de cada faixa

FaixaCriterio
estavelTeste versionado compila e linka no CI. Comportamento documentado cobre casos de borda conhecidos.
parcialImplementacao existe mas tem lacunas documentadas. Testes podem ser parciais ou faltar.
experimentalImplementacao existe e funciona em casos tipicos. API pode mudar. Testes minimos.
planejadaNada implementado. Apenas especificacao preliminar ou discussao.
