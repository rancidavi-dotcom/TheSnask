Testes da documentacao

Exemplos de documentacao so contam como teste real quando existem como arquivo `.snask` versionado e passam por `snask build`.

Rodar tudo

scripts/check_doc_examples.sh

Onde ficam os exemplos

Os exemplos buildaveis ficam em docs/examples/reference/. Cada pagina de funcao que possui exemplo validado mostra o comando exato para compilar aquele arquivo.

Politica de honestidade

  • Se compila e linka hoje, a pagina pode apontar para um teste real.
  • Se existe apenas no analisador semantico, a pagina fica marcada sem teste versionado.
  • Se depende de GUI ou ambiente externo, o exemplo deve explicar a dependencia.
  • Se a feature e parcial ou experimental, a pagina deve dizer isso antes do exemplo.
