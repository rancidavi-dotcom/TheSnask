Partes (lo / hi / make)

Funções para manipular partes de words de 16 bits, usadas em programação de sistemas e emuladores NES.

FunçãoDescrição
lo_u8(word)Extrai o byte baixo (bits 0-7) de uma word u16
hi_u8(word)Extrai o byte alto (bits 8-15) de uma word u16
make_u16(lo, hi)Monta uma word u16 a partir de dois bytes u8
 
let word: u16 = 0xABCD
let lo = lo_u8(word)    // 0xCD
let hi = hi_u8(word)    // 0xAB
let reconstruida = make_u16(lo, hi)  // 0xABCD

Memória Crua

Funções de acesso direto à memória. Requerem @unsafe porque podem invalidar as garantias de segurança de memória da Snask.

FunçãoDescriçãoRequer
mem_alloc(size)Aloca bloco de memória do tamanho especificado@unsafe
mem_alloc_zero(size)Aloca bloco zerado@unsafe
mem_free(ptr)Libera bloco alocado manualmente@unsafe
mem_copy(dst, src, n)Copia n bytes de src para dst@unsafe
mem_fill_u8(ptr, val, n)Preenche n bytes com val@unsafe
mem_read_u8(ptr, offset)Lê um byte no offset@unsafe
mem_read_u16(ptr, offset)Lê uma word (little-endian) no offset@unsafe
mem_read_u32(ptr, offset)Lê uma dword (little-endian) no offset@unsafe
mem_write_u8(ptr, offset, val)Escreve um byte no offset@unsafe
mem_write_u16(ptr, offset, val)Escreve uma word (little-endian) no offset@unsafe
mem_write_u32(ptr, offset, val)Escreve uma dword (little-endian) no offset@unsafe
ptr_add(ptr, offset)Desloca o ponteiro por offset bytes@unsafe
 
@unsafe {
    let ptr = mem_alloc(64)
    mem_write_u8(ptr, 0, 0xFF)
    let val = mem_read_u8(ptr, 0)   // 0xFF
    mem_free(ptr)
}

O módulo sfs (Snask File System) oferece operações avançadas de sistema de arquivos. Deve ser importado com import "sfs".

FunçãoDescrição
sfs::read(path)Lê arquivo como string
sfs::write(path, data)Escreve string em arquivo
sfs::append(path, data)Adiciona ao final do arquivo
sfs::copy(src, dst)Copia arquivo

sfs::delete(path)Remove arquivo

sfs::is_dir(path)Verifica se é diretório

SnaskGUI (Experimental)

SnaskGUI é uma alternativa leve à GUI baseada em GTK, voltada para renderização de gráficos pixel-a-pixel (útil para emuladores e ferramentas gráficas simples).

FunçãoDescrição
snaskgui_init()Inicializa o subsistema SnaskGUI
snaskgui_window(w, h, title)Cria janela com largura e altura
snaskgui_poll()Processa eventos pendentes
snaskgui_delay(ms)Atraso em milissegundos
snaskgui_should_close()Verifica se a janela deve fechar
snaskgui_present_rgba(data, w, h)Renderiza buffer RGBA na janela
snaskgui_close()Fecha a janela
snaskgui_key_down(key)Verifica se uma tecla está pressionada
 
snaskgui_init()
let win = snaskgui_window(256, 240, "Emulador")

let framebuffer: list<u8> = []
// preencher framebuffer com pixels RGBA...

while not snaskgui_should_close() {
    snaskgui_poll()
    snaskgui_present_rgba(framebuffer, 256, 240)
    snaskgui_delay(16)

    if snaskgui_key_down("escape") {
        snaskgui_close()
    }
}

experimental SnaskGUI para renderização gráfica.
