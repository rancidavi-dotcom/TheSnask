Runtime e builtins

Builtins sao funcoes fornecidas pelo compilador/runtime. Algumas existem em `humane`, outras so fazem sentido no perfil `systems` e algumas exigem `@unsafe`.

println()humaneQuando disponivel, imprime com quebra de linha.

Memoria crua

Funcoes de memoria crua sao para systems programming. Elas ficam protegidas por `@unsafe` porque podem invalidar garantias comuns de memoria.

FuncaoUso
mem_alloc(size)Aloca bloco cru.
mem_alloc_zero(size)Aloca bloco cru zerado.
mem_free(ptr)Libera bloco alocado manualmente.
mem_read_u8(ptr, offset)Le byte em offset.
mem_write_u8(ptr, offset, value)Escreve byte em offset.
mem_read_u16(ptr, offset)Le word little-endian.
mem_write_u16(ptr, offset, value)Escreve word little-endian.
