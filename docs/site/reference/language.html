Arquivo executavel

Um executavel usa `class main` e o metodo `start` como entrada recomendada.

class main {
    fun start() {
        print("ok\n")
    }
}

Blocos

Snask aceita blocos por indentacao e por chaves.

if ready {
    print("ready\n")
}
if ready
    print("ready\n")

Operadores

Aritmetica comum, comparacoes, operadores logicos, bitwise e shifts existem. Semantica de igualdade/comparacao mista ainda esta em consolidacao. 

let a = 10 + 2
let b = a & 0xFF
let ok = a >= b and b != 0

Modulos

import "sfs"
from utils import math_helpers

Imports simples sao estaveis para arquivos e modulos nativos. `from ... import ...` ainda e mais simples e deve amadurecer.
