SPS e SNIF

SPS e o sistema de projeto. SNIF e o formato usado pelo manifesto `snask.snif`.

Manifesto

{
  package: { name: "store", version: "0.1.0", entry: "main.snask" },
  dependencies: { json: "*" },
  build: { profile: "humane", opt_level: 2 },
  scripts: { run: "snask run" },
}

Comandos SNIF

snask snif fmt snask.snif --write
snask snif schema --md
snask snif check snask.snif

Lockfile

`snask.lock` registra pacote, registry e dependencias com `version`, `sha256` e URL opcional. Isso ja tem roundtrip testado.
