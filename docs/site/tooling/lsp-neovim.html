Instalação CLI SPS e SNIF Neovim
Opcional: esta página é apenas para quem deseja usar o Neovim como editor. O Snask funciona com qualquer editor — o LSP e o plugin são opcionais.

Editor

Configuração do Neovim

Instale o servidor LSP do Snask e o plugin snask.nvim para ter syntax highlighting, indentação, diagnósticos e comandos integrados.

Arch Linux

O pacote AUR snask instala apenas o compilador. O LSP deve ser compilado separadamente.

1. Compilar o LSP

# A partir do diretório do repositório Snask
cargo build --release --bin snask-lsp

# Copiar para a pasta do Snask
mkdir -p ~/.snask/bin
cp target/release/snask-lsp ~/.snask/bin/

# Adicionar ao PATH (adicione no ~/.bashrc ou ~/.zshrc)
export PATH="$HOME/.snask/bin:$PATH"

2. Instalar o Neovim

sudo pacman -S neovim

3. Instalar o plugin snask.nvim

O plugin está incluso no repositório do Snask em editors/neovim/snask.nvim. Crie um link simbólico para o diretório de plugins do Neovim:

mkdir -p ~/.local/share/nvim/site/pack/snask/start

# Ajuste o caminho abaixo para onde você clonou o repositório
ln -s /caminho/para/TheSnask/editors/neovim/snask.nvim \
  ~/.local/share/nvim/site/pack/snask/start/snask.nvim

4. Verificar

nvim --headless -c "checkhealth snask" -c "q"

Linux Mint (Ubuntu/Debian)

No Linux Mint, instale o Snask pelo .deb ou compile manualmente, depois compile o LSP separadamente.

1. Compilar o LSP

# A partir do diretório do repositório Snask
cargo build --release --bin snask-lsp

# Copiar para a pasta do Snask
mkdir -p ~/.snask/bin
cp target/release/snask-lsp ~/.snask/bin/

# Adicionar ao PATH (adicione no ~/.bashrc ou ~/.zshrc)
export PATH="$HOME/.snask/bin:$PATH"

2. Instalar o Neovim

sudo apt update
sudo apt install neovim
AppImage: se o repositório do Mint tiver uma versão antiga do Neovim, baixe o AppImage da página oficial: github.com/neovim/neovim/releases.

3. Instalar o plugin snask.nvim

mkdir -p ~/.local/share/nvim/site/pack/snask/start

# Ajuste o caminho abaixo para onde você clonou o repositório
ln -s /caminho/para/TheSnask/editors/neovim/snask.nvim \
  ~/.local/share/nvim/site/pack/snask/start/snask.nvim

4. Verificar

nvim --headless -c "checkhealth snask" -c "q"

Comandos do plugin

Com o plugin instalado, os seguintes comandos ficam disponíveis no Neovim:

ComandoDescrição
:SnaskBuildCompila o projeto atual
:SnaskRunExecuta o binário compilado
:SnaskDoctorExecuta snask doctor
:SnaskExplain S1002Explica um código de diagnóstico
:SnaskOmScanEscaneia bibliotecas C para OM
:checkhealth snaskVerifica se o LSP e o plugin estão OK
