Ferramentas

CLI

A CLI `snask` e a entrada principal para compilar arquivos, projetos SPS, benchmarks locais e experimentos systems.

Build de arquivo

snask build main.snask --output app
./app

Sem `--profile`, o compilador usa `humane`.

Build com perfil

snask build apps/nes_emulator/nes_master.snask --profile systems --output /tmp/snask_nes

Projetos SPS

snask setup
snask build
snask run

Flags de tamanho

FlagUso
--release-sizeBuild otimizado para tamanho.
--min-runtimeRemove partes opcionais do runtime.
--tinyModo mais agressivo para binarios pequenos.
--extremeExperimento de tamanho minimo, com restricoes maiores.
Use `docs/reference/FEATURE_STATUS.md` como fonte de verdade quando uma flag ainda estiver marcada como parcial ou experimental.
