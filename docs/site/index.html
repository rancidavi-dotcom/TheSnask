Snask v0.4.1-alpha

Uma linguagem compilada, humana por padrao, systems quando necessario.

Snask compila AOT para binarios nativos via LLVM. O objetivo e unir uma sintaxe clara, diagnosticos humanos, memoria orquestrada pelo OM-Snask-System e acesso controlado a baixo nivel.

O que a Snask e hoje

Compilada de verdade

O compilador le `.snask`, constroi AST, analisa semantica, gera LLVM IR e chama o linker. Snask nao e transpiler para C.

Humane por padrao

Sem `--profile`, a linguagem usa o perfil `humane`, com runtime completo e diagnosticos mais legiveis.

Systems quando precisa

O perfil `systems` expoe inteiros exatos, operacoes de bits e memoria crua protegida por `@unsafe`.

OM-Snask-System

Zonas, recursos externos, contratos C deduzidos e limpeza deterministica pertencem ao mesmo sistema de memoria.
