Snask é uma linguagem compilada AOT (Ahead-of-Time) via LLVM 18 que une tipos estáticos modernos com controle total sobre o hardware. Desenvolva desde scripts rápidos até emuladores — tudo na mesma linguagem.

🚀 Compilado Nativo 🧠 Memória Determinística 🛡️ Type-Safe 🎭 Perfis Humane + Systems
Começar a Aprender Referência de API GitHub
⚡

Compilado via LLVM 18

Gera binários nativos ultra-otimizados. Sem VM, sem interpretador. Performance comparável a C e Rust.

🧠

OM-Snask-System

Memória determinística via Zonas. Performance de C com segurança de tipos, sem pausas de garbage collector.

🎭

Dois Perfis, Uma Linguagem

Use Humane para apps (GUI, HTTP, SQLite) ou Systems para bits, ponteiros e hardware.

🔧

Toolchain Completo

Compilador, formatador SNIF, sistema de pacotes (SPS), LSP, REPL, empacotamento .deb e AUR integrados.

🌐

Interop com C

Importe bibliotecas C nativamente via contratos OM. SQLite, zlib, GTK, Skia — tudo disponível sem bindings manuais.

📦

Binários Pequenos

Compile com --tiny ou --extreme para binários abaixo de 100KB. Ideal para embedded e CLI.

Sintaxe Limpa e Expressiva

class main {
    fun start() {
        zone "frame" {
            let user = "Davi"
            let score = 100
            print("Olá, {user}! Score: {score}\n")
        }
    }
}
import "math"

fun fibonacci(n: int) : int {
    if n <= 1 { return n }
    return fibonacci(n - 1) + fibonacci(n - 2)
}

print("{fibonacci(10)}\n")  // 55
