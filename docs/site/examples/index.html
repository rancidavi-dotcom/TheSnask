Exemplos

Exemplos pequenos e copiaveis para aprender a superficie atual da Snask.

CLI simples

class main {
    fun start() {
        let name = "Snask"
        print("Ola, {name}!\n")
    }
}

Leitura de arquivo via modulo nativo

import "sfs"

class main {
    fun start() {
        let text: str = sfs::read("README.md")
        print(text)
    }
}

Flags de CPU

class main {
    fun start() {
        let flags: u8 = flag_write(0, 7, true)
        let negative: bool = flag_has(flags, 7)
        print("N={negative}\n")
    }
}

Memoria crua

class main {
    fun start() {
        @unsafe {
            let mem: ptr = mem_alloc_zero(256)
            mem_write_u8(mem, 0, 42)
            let value: u8 = mem_read_u8(mem, 0)
            mem_free(mem)
            print("Valor: {value}\n")
        }
    }
}
