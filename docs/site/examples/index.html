Exemplos Snask — Cookbook

Exemplos pequenos e copiaveis para aprender a superficie atual da Snask. Cada exemplo pode ser compilado com snask build. Consulte o Status Real para saber o nivel de maturidade de cada area.

# CLI simples

Programa minimo que imprime uma saudacao com interpolacao de string:

class main {
    fun start() {
        let name = "Snask"
        print("Ola, {name}!\n")
    }
}
snask build hello.snask && ./hello
ConceitoDemonstrado
class mainPonto de entrada do programa
fun start()Metodo inicial chamado pelo runtime
letVariavel imutavel
print()Saida padrao com interpolacao

# Funcao com retorno

Funcoes tipadas com parametros e retorno. O tipo de retorno vem depois dos ::

class main {
    fun soma(a: i32, b: i32): i32 {
        return a + b
    }

    fun start() {
        let r = soma(10, 20)
        print("soma = {r}\n")
    }
}
ConceitoDemonstrado
fun nome(a: T): RFuncao com parametro e retorno tipados
returnRetorno de valor
Inferencia de tipolet r = soma(...) infere i32

# Classe com metodos e heranca

Classes nominais com propriedades, metodos e heranca simples. Uma subclasse pode ser usada onde a classe pai e esperada:

class Animal {
    let nome: str

    fun new(n: str): Animal {
        return Animal { nome: n }
    }

    fun falar(self) {
        print("{self.nome} faz: ...\n")
    }
}

class Cachorro : Animal {
    fun falar(self) {
        print("{self.nome} faz: Au Au!\n")
    }
}

class main {
    fun start() {
        let a: Animal = Cachorro::new("Rex")
        a.falar()
    }
}
ConceitoDemonstrado
class Nome : BaseHeranca simples
fun new(...): NomeConstrutor nominal
Animal { campo: valor }Inicializador de struct
selfReceptor de metodo
SubtipoCachorro pode ser atribuido a Animal

# Lista e dicionario

Colecoes genericas homogeneas com literais, indexacao e atribuicao:

class main {
    fun start() {
        let nums: list<i32> = [1, 2, 3, 4]
        nums[2] = 99
        print("{nums[2]}\n")

        let mapa: dict<str, i32> = {
            "um":   1,
            "dois": 2,
            "tres": 3
        }
        print("{mapa["dois"]}\n")
    }
}
ConceitoDemonstrado
list<T>Lista generica homogenea
dict<K, V>Dicionario generico homogeneo
lista[indice]Indexacao e atribuicao
{"chave": valor}Literal de dicionario

# Flags de CPU (bit operations)

Operacoes de bits e flags do perfil Systems. Exige estar dentro do perfil correto:

class main {
    fun start() {
        let flags: u8 = flag_write(0, 7, true)
        let negative: bool = flag_has(flags, 7)
        print("N={negative}\n")

        let masked: u8 = flags & 0b1000_0000
        print("masked={masked}\n")
    }
}
ConceitoDemonstrado
flag_writeEscreve um bit em posicao
flag_hasTesta se um bit esta ligado
&Operador bitwise AND

# Memoria crua (bloco @unsafe)

O perfil Systems permite acesso a memoria crua dentro de blocos @unsafe. Fora deles, o compilador bloqueia o uso:

class main {
    fun start() {
        @unsafe {
            let mem: ptr = mem_alloc_zero(256)
            mem_write_u8(mem, 0, 42)
            let value: u8 = mem_read_u8(mem, 0)
            mem_free(mem)
            print("Valor: {value}\n")
        }
    }
}
O uso de @unsafe desliga garantias de memoria para o bloco. Prefira o sistema OM sempre que possivel.
ConceitoDemonstrado
@unsafe { ... }Bloco de memoria nao segura
mem_alloc_zeroAloca memoria inicializada com zero
mem_write_u8 / mem_read_u8Acesso a byte na memoria
mem_freeLibera memoria alocada

# Uso de zona OM

O Object Memory system gerencia alocacao por zonas. Recursos sao registrados e liberados automaticamente ao final da zona:

import "om"

class main {
    fun start() {
        om::zone_new("leitura")
        let arquivo = om::resource_open("dados.txt")
        let dados = om::resource_read(arquivo)
        om::zone_end("leitura")
        print(dados)
    }
}
OM e experimental. A API pode mudar. Consulte Status Real para detalhes.

# Interpolacao de strings

Strings aceitam interpolacao de variaveis e tipos primitivos dentro de chaves:

class main {
    fun start() {
        let nome = "Snask"
        let versao = "0.1"
        let ano = 2025
        print("Bem-vindo ao {nome} v{versao} ({ano})\n")
    }
}
ExpressaoResultado
"ola {nome}"Interpola variavel nome
"v{versao}"Interpola string adjacente
"({ano})"Interpola inteiro

# HTTP GET request

O modulo nativo http permite requisicoes HTTP simples:

import "http"

class main {
    fun start() {
        let res: str = http::get("https://api.example.com/data")
        print(res)
    }
}
O modulo HTTP requer conectividade de rede. Exemplos que dependem de rede externa sao marcados com # skip-ci e nao bloqueiam o pipeline.

# JSON parsing

O modulo nativo json permite parsear e consultar dados JSON:

import "json"

class main {
    fun start() {
        let dados = json::parse('{"nome": "Snask", "ano": 2025}')
        let nome = json::get(dados, "nome")
        let ano = json::get(dados, "ano")
        print("{nome} ({ano})\n")
    }
}
FuncaoDescricao
json::parse()Converte string JSON para objeto interno
json::get(objeto, chave)Acessa campo do objeto
json::stringify()Converte objeto de volta para string
json::has(objeto, chave)Verifica se chave existe
